# StalacWorld

StalacWorld je 2D příběhová hra vytvořená během hackathonu v Unity.  

Hráč se ujímá role Marie, která prochází lesem, jenž se postupně mění v jiný svět zvaný StalacWorld. Pomáháním přátelským monstrům získává jejich podporu, aby se mohla vrátit zpět do svého světa.

Hra zatím obsahuje pouze 1 quest. Další části (Ruins) budou přidány v budoucnu.

## Použité technologie
- Unity (2D)
- C#
- Git a GitHub pro verzování a týmovou spolupráci

## Funkce
- 2D průchod světem StalacWorld
- Interakce s přátelskými NPC (monstry)
- Příběhová mechanika s questem
- Základní systém pomoci od NPC

## Stav projektu
- Early version / Hackathon build
- Plně funkční 1 quest
- Další části budou postupně doplňovány
